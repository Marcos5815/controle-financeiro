# Finances

Observação: A aplicação utiliza o plano gratuito do Supabase. Caso o projeto fique algum tempo sem acesso, a primeira requisição pode levar alguns segundos devido ao cold start.

## Instalação

- npm install

## Sobre o Projeto

Essa é uma aplicação de controle financeiro desenvolvida para auxiliar usuários no gerenciamento de receitas e despesas, fornecendo visualizações gráficas das despesas por categoria, filtros por período e personalização das categorias e métodos de pagamento.

## Funcionalidades*

- Autenticação segura com Clerk
- Persistência dos dados com o supabase
- Adição, edição e remoção de transações financeiras
- Controle de receitas e despesas
- Filtro de transações por data
- Visualização das transações em lista
- Gráficos para análise de despesas
- Suporte aos idiomas Português e Inglês
- Tema claro e escuro

## Como Utilizar

- Crie uma conta utilizando um e-mail válido ou faça login em uma conta existente.
- Após o login, será direcionado ao dashboard principal.
- Adicione receitas e despesas através do formulário de transações.
- Altere as categorias das transações ou métodos de pagamentos para se adequar a sua realidade.
- Visualize todas as movimentações financeiras na lista de transações.
- Utilize os filtros por data para analisar períodos específicos.
- Consulte os gráficos para acompanhar a distribuição e evolução das despesas.
- Personalize a experiência alterando o idioma entre Português e Inglês.
- Escolha entre tema claro e escuro nas configurações.
